Serif AJAX Interface

This sample interface uses jquery to construct a SerifXMLRequest and send it to the Serif HTTP server. It then displays the SerifXML result.

Input Document

Type or copy/paste an input document into the text area below, and select options (including the desired end stage). Then click "Run Serif" to process the document with this server.
