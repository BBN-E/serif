Serif Output Formats

The Serif HTTP server supports the following output formats:

SerifXML This is the default output format. It is an XML file that includes all of SERIF's annotation information for the document. The possible contents for this file are defined by the SerifXML Document Type Definition (DTD).
APF, APF4, APF5, APF7, APF8 The "ACE Pilot Format". This format was defined as part of the Automatic Content Extraction (ACE) Evaluation, as a common format that could be used to compare information extraction system performance. It is recommended that you use APF8 (the most recent version). Document Type Definitions (DTDs): APF5, APF7, APF8
PARSE A simple text output format that includes only the output of the parser.
