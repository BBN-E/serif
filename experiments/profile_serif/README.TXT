The output files generated by "callgrind" are best viewed using the
kcachegrind tool, which is available at the following locations:

KCacheGrind for Windows:
http://sourceforge.net/projects/precompiledbin/files/kcachegrind.zip/download

KCacheGrind for Linux:
/usr/bin/kcachegrind

Cycle Estimation:
CEst = Ir + 10 L1m + 100 LLm

# Bm = Bim + Bcm -- branch prediction
# Ge = global events (?)
CEst = Ir + 10 Bm + 10 L1m + 20 Ge + 100 L2m + 100 LLm
